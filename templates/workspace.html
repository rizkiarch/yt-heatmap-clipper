🔐 YouTube Authentication

YouTube memblokir download tanpa login. Jika Anda melihat error "Sign in to confirm you're not a bot", berikut 3 cara untuk mengatasinya (dari yang paling direkomendasikan):

1️⃣ Cookies dari Browser (Paling Direkomendasikan)

1. Install ekstensi Chrome Get cookies.txt LOCALLY
2. Buka youtube.com dan pastikan Anda sudah login
3. Klik ikon ekstensi → Export → pilih Netscape format → simpan sebagai cookies.txt di folder project ini
4. Restart aplikasi dan coba lagi

2️⃣ Baca Cookies Langsung dari Browser

Atur environment variable di file .env: YTDLP_COOKIES_FROM_BROWSER=chrome (atau firefox, edge, safari)
Pastikan browser sudah login ke YouTube sebelum menjalankan aplikasi.

3️⃣ PO Token + Visitor Data

Jika 2 cara di atas tidak berhasil:
1. Buka youtube.com di browser, tekan F12 → tab Network
2. Mainkan video apa saja, cari request yang namanya player
3. Di payload, cari poToken dan visitorData
4. Tambahkan ke .env: YTDLP_PO_TOKEN=token_anda dan YTDLP_VISITOR_DATA=data_anda
